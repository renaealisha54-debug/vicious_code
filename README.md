# ViciousLayer

ViciousLayer is an Android system overlay utility designed for high-performance background operations.

## Project Architecture
This project is built using Kotlin and follows the standard Android Gradle project structure:
- **Build System**: Managed via Gradle Wrapper for consistent builds.
- **Project Structure**: Modularized with a root configuration and a specific `app` module.

## Getting Started
To build this project:
1. Ensure you have the Android SDK installed.
2. Clone the repository.
3. Run the build command: `./gradlew assembleDebug`
